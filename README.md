# Klaropoint Marketing Website

Premium Next.js 15 marketing site for Klaropoint.

## Run locally

```bash
npm install
npm run dev
```

## Contact form

Copy `.env.example` to `.env.local`, add a Resend API key, and verify the sender domain. The form posts to `/api/contact`.

## Deploy

Deploy directly to Vercel or any platform supporting Next.js 15. Add the environment variables in the hosting dashboard.
